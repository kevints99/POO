Descripción

Este programa define una clase Libro que representa un libro con atributos como título, autor y número de páginas. También proporciona métodos para mostrar información del libro y actualizar la cantidad de páginas.

Estructura del Código

El código contiene lo siguiente:

Clase Libro:

__init__(self, titulo, autor, num_paginas): Constructor que inicializa el título, el autor y el número de páginas del libro.

mostrar_informacion(self): Método que imprime en pantalla la información del libro.

actualizar_paginas(self, nuevas_paginas): Método que actualiza el número de páginas del libro y lo muestra en pantalla.

Ejemplo de uso:

Se crea una instancia de la clase Libro con datos de "El Quijote".

Se muestra la información inicial del libro.

Se actualiza el número de páginas y se vuelve a mostrar la información actualizada.
